Overview

This project implements a modular recruitment evaluation system written in C++. The system ingests structured input data for human and alien recruits, validates and corrects malformed entries, and stores records using custom singly linked lists. It provides efficient search, deletion, and merge operations without relying on standard library containers or sorting utilities.

Design and Implementation
  • Abstract base class CrewMember defining shared attributes and comparisons
  • Human and Alien classes derived using inheritance
  • TelepathicLinkTest composed within Alien recruits for qualification logic
  • RecruitmentSystem class coordinating parsing, validation, and menu logic
  • Clear separation of headers and source files across the codebase
Data Structures and Algorithms
  • Custom singly linked lists with explicit head and tail pointers
  • Sorted insertion performed in O(n) time at insertion
  • Search by ID, training score, mission aptitude, and full name
  • Deletion by name including head and tail edge cases
  • Merge operation combining human and alien lists with deterministic ordering
Error Handling and Validation
  • Detection of missing or malformed input fields
  • Case insensitive string comparisons throughout the system
  • Automatic correction of known data entry errors with warnings
  • Graceful handling of invalid menu selections
  • Exception handling during memory allocation and input failures
Unit Testing
  • Verification of sorted insertion behavior
  • Search correctness across all supported fields
  • Deletion correctness including boundary cases
  • Merge operation correctness and stability
  • Qualification filtering based on TelepathicLinkTest thresholds
Algorithmic Complexity
  • Insertion: O(n)
  • Search: O(n)
  • Deletion: O(n)
  • Merge: O(n)
Tech Stack
  • Language: C++
  • Paradigm: Object Oriented Programming
  • Data Structures: Custom singly linked lists
  • Testing: Manual unit testing framework
